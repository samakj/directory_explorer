# Directory explorer

```
usage: run.py [-h] [-f FOLDER] [-d DEPTH] [-sh SHOW_HIDDEN] [-e EXCLUDE]

optional arguments:
  -h, --help            show this help message and exit
  
  -f FOLDER, --folder FOLDER
                        The directory to explore.
  -d DEPTH, --depth DEPTH
                        How many levels to traverse.
  -sh SHOW_HIDDEN, --show_hidden SHOW_HIDDEN
                        Show hidden files.
  -e EXCLUDE, --exclude EXCLUDE
                        Folder name to exclude. Repeat flag for multiple
                        folders.
```
